# Hero images — clean motorcycle photos

Place clean, high-quality motorcycle photographs here for the hero carousel.

## Requirements

- No text, logos, prices, borders, or commercial information embedded in the image
- Aspect ratio: 16:9 recommended (will be cropped to 16:10 on mobile)
- Background: preferably solid/studio or open road (no busy backgrounds)
- Format: WebP preferred, JPEG fallback
- Size: ~1200×675px or larger
- File naming: `moto-1.webp`, `moto-2.webp`, etc.

## Current fallback

Until clean photos are placed here, the carousel uses existing `img/moto-*.jpg` files
from the catalog. These are clean motorcycle photographs without commercial overlays.

The old `img/hero.jpg` (a promotional composite with text, logos, and frames) has been
removed from the carousel array — it remains on disk for potential use in the catalog
but is no longer shown in the hero.
